⊙ RITUAL ⊙

Hermetic LLM Context Management Portal

A beautiful, performant local portal for managing MCM files and LLM connections with mystical aesthetics — built on FastAPI, running entirely offline.

⊕ Get Started View on GitHub

✨ Key Features

⊙

One-Button Interface

Simplify your workflow with elegant, focused simplicity. Everything you need, nothing you don't.

🔮

Mystical Aesthetic

Unique hermetic visual design that elevates LLM context management into an experience.

⚡

60fps Animations

Smooth, buttery UI transitions powered by CSS3 and Vanilla JavaScript.

🔒

Local-Only Design

No cloud dependencies. Your data never leaves your machine. Full privacy guaranteed.

🛡️

Encrypted Storage

API keys and sensitive data are encrypted at rest using Fernet symmetric encryption.

🐍

Python-Powered

Built on FastAPI for reliability, speed, and a clean REST API you can build on.

🚀 Quick Start

1

Clone & Install

git clone https://github.com/black-tiger-computing/ritual.git
cd ritual
pip install -r requirements.txt
2

Start the Server

python -m src.backend.main
3

Open the Portal

http://localhost:8000

For detailed setup instructions, see the Installation Guide.
